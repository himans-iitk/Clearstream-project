## Requirements:
1. Django

```pip install django```

2. django-crispy-forms

```pip install django-crispy-forms```

3. Pillow

```pip install pillow```

## Running the project for development:
```
python manage.py makemigrations
python manage.py migrate
python manage.py runserver <port>

```
